INTRODUCTION

CloudSky is a Multipurpose Domain, Hosting and WHMCS Template designed for All kinds of Technology, Domain and Hosting Business. It's an HTML5 template based on Bootstrap 3.3.7. Also included WHMCS Template based on WHMCS v7.5.1. Anyone can easily update/edit this plugin to follow our Well Sorted Documentation.

Mainly, our template is an HTML template and we just provided support by customizing WHMCS header and footer and color based on our template for FREE of cost with our HTML template.

Main Features of "CloudSky":

  • - Stable Bootstrap v3.3.7 Ready
  • - Fully Responsive

  • - Google Font Used

HTML STRUCTURE

The "CloudSky" template is a responsive HTML template based on Bootstrap 3.3.7. The structure based on a flexible grid that lets you change HTML pages quickly and logically with a nestable system.

If you would like to edit the layout of any elements in whole template, you would do the following:

Open any '.html' or, '.css', so that is extremely easy to change any html markup and css attributes. Simply find the related and commented heading in the html or, css file and make your magic.

INSTALLATION

Via FTP:

1. Open up your 'FTP Software' of choice and connect to your hosting

2. Browse to required directory (Normally: ./public_html)

3. Unzip the 'main_files.zip' and choose which templates to upload.

4. Upload the template to the server.

FAQ

Once you finished setup successfully, you can customize it as you want. Below are some common frequently asked questions.

Q. How To Change Theme Color Scheme?

For HTML Template:

  • Open '.html' file. (For Example: 'index.html')
  • Find the comment 'Theme Color Stylesheet'
  • Update the file name 'theme-color-1.css' to 'theme-color-x.css' or change/update the color code with your desire/brand color code in the 'theme-color-1.css' file.

Available Color Schemes:

For WHMCS Template:

  • Open 'head.tpl' file from the includes folder.
  • Update the file name 'theme-color-1.css' to 'theme-color-x.css' or change/update the color code with your desire/brand color code in the 'theme-color-1.css' file.

Q. How To Change Default Color Schemes?
